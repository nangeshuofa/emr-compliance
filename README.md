# 电子病历合规画像

> 作者：楠哥说法（黄楠律师）
> 版本：v1.0.0
> 协议：GPL-3.0
> 语言：简体中文

基于10大维度、74个检查点的电子病历合规评估工具，通过交互式提问对医疗机构的电子病历系统进行全面合规体检，生成专业的合规画像报告。

---

## ✨ 功能特点

| 特点 | 说明 |
|------|------|
| 🎯 **全面覆盖** | 10大维度、74个检查点，全方位评估电子病历合规性 |
| 📊 **可视化报告** | 雷达图评分 + 风险等级 + 整改建议，一目了然 |
| ⚖️ **法律依据充分** | 每个检查点都有明确的法律法规依据 |
| 🔧 **可操作建议** | 整改建议具体可落地，不是空泛的"建议完善" |
| 🤝 **交互友好** | 自然语言提问，不是机械的逐条审问 |

## 🚀 快速开始

### 方式一：作为 AI 技能使用

将 `emr-compliance-profile.skill` 导入到支持 Skill 的 AI 平台，然后对助手说：

```
帮我做电子病历合规评估
```

按照提示回答问题，完成后自动生成合规画像报告。

### 方式二：本地使用

克隆本仓库，按照 `SKILL.md` 中的评估流程，对照 `02_评估维度/` 下的检查点进行自查。
