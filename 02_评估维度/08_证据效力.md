# 维度8：证据效力（6项）

> 电子病历最终都是要上法庭的。证据效力不够，写得再漂亮也没用。

## 法律依据

### 核心法规
- **《中华人民共和国电子签名法》**
  - 第7条：数据电文不得仅因为其是以电子、光学、磁或者类似手段生成、发送、接收或者储存的而被拒绝作为证据使用
  - 第8条：审查数据电文作为证据的真实性，应当考虑的因素
  - 第14条：可靠的电子签名与手写签名或者盖章具有同等的法律效力

- **《中华人民共和国民事诉讼法》**
  - 第66条：证据包括：...（八）视听资料；（九）电子数据
  - 第79条：当事人可以就查明事实的专门性问题向人民法院申请鉴定

- **《最高人民法院关于民事诉讼证据的若干规定》**
  - 第23条：人民法院调查收集视听资料、电子数据，应当要求被调查人提供原始载体

### 证据真实性审查标准（电子签名法第8条）

法院审查电子数据真实性时，主要看：
1. **生成方法的可靠性**：系统是否可靠，数据是怎么生成的
2. **存储方法的可靠性**：数据存在哪里，存储方式是否安全
3. **内容完整性**：数据有没有被篡改过，能不能证明完整性
4. **发件人鉴别方法**：能不能证明是谁操作的

### 常见风险
1. **审计日志不完整**：只有修改人，没有修改前后内容对比
2. **签名无法验证**：电子签名是医院自己做的，第三方不认
3. **打印件与系统不一致**：纸质病历和电子病历对不上
4. **不知道去哪鉴定**：真要打官司了，不知道找哪家机构做电子数据鉴定

### 评估要点
- 如果打官司，你们的电子病历能作为证据使用吗？
- 审计追踪能不能导出成独立文件？
- 电子签名有没有第三方CA机构的认证？
- 有没有和司法鉴定机构打过交道？
- 有没有遇到过法庭要求出示电子病历原件的情况？
